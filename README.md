# reseptikanta

Simple collection of recipes.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Push to gh-pages

(https://www.youtube.com/watch?v=yo2bMGnIKE8)

```sh
npm run build
git add dist -f
git commit -m "..."
git subtree push --prefix dist origin gh-pages
```
